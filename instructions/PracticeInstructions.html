Practice Game Instructions


To get familiar with the task, you will now get to play 3 practice rounds with four other players.


The practice game is set up in the same way as the main game, but there are no reversals, and the reward probabilities are fixed. As such, you should not learn anything about the reward probabilities from the practice game.


This is just to get familiar with how the task works: which buttons to press and when.


The time provided for each of the individual sections is also longer in the practice game (approximately 8 seconds per decision), compared to the actual game (between 2-3 seconds per decision), so you can understand the different components and have more time to make your decisions.


Please press the next button when you are ready to play the practice game. We will put you into a waiting room where you will be matched with four other players.

It should take no longer than a few minutes to be matched (which is factored into your payment).


The practice task itself will last approximately 3 minutes. Please also change the size of your browser window to match the page if needed.
